Ruby Enumerable Cheat Sheet

Source: Ruby Docs

Method Name Effect Example Output
.include?() Returns true if any element == value "hello world".include?("hello") => true
.all? Returns true if block never returns “false”. [1,2,3].all? { |x| x < 10 } => true
.sort Returns array sorted by <=> operator or by block. [1,2,3].sort { |a,b| b <=> a } => [3,2,1]
.sort_by Returns array sorted by the return value of the block. %w{apple pear fig}.sort_by { |word| word.length} ["fig", "pear", "apple"]
.find_index Returns index of the first element where block returns “true” (1..100).find_index { |x| x % 5 == 0 and x % 7 == 0 } => 34
.take() Returns array of first n elements. [1,2,3].take(2) => [1,2]
.drop_while Returns array of all elements starting with the one where the block returned “false”. [1,2,3].drop_while { |i| i < 2 } => [1,2]
.each_slice Iterates the given block for each slice of elements 1..6).each_slice(3) { |a| p a } [1, 2, 3]
[4, 5, 6]
.group_by Returns a hash associating block return values to arrays of elements. (1..6).group_by { |i| i % 3 } => {0=>[3, 6], 1=>[1, 4], 2=>[2, 5]}
.partition Returns [true_array, false_array] (1..6).partition { |v| v.even? } => [[2, 4, 6], [1, 3, 5]]
.map/.collect Returns an array with the results from block. [1,2,3].map { |num| num * 2 } => [2,4,6]
.each_with_index Yields all elements with their indices. args are passed through to #each [1,2,3,4].each_with_index { |num, index| puts num * index } => 0,2,6,12
.zip Merges enum with other enums to make arrays that each have an element from each enum; same length as original enum. a = [ 4, 5, 6 ] b = [ 7, 8, 9 ]
[1, 2, 3].zip(a, b)
[1, 2].zip(a, b) 		=> [[1, 4, 7], [2, 5, 8], [3, 6, 9]]
=> [[1, 4, 7], [2, 5, 8]]
