Intro to Regular Expressions
15 December 2014

Regular expressions can be very useful in your programming career and having a good understanding of them will be essential. A regular expression is used to match a pattern against strings. Also called Regexps, they are created using the /.../ or the Regexp::new constructor. Regular expression are basically code using weird symbols to symbolize characters. Regular expressions are patterns which describe the contents of a string. They’re used for testing whether a string contains a given pattern, changing or extracting the portions that match. Here are some examples:

string = “hi, how are you?”
string.sub(/^../, “hey”)
>// “hey, how are you?”

All regex start with / / so the stuff inside the / / will be executed. Above, the ^ refers to the beginning of the string and the dots represent the number of characters. Therefore, this sub method will replace the first two characters in the string, “hi” with “hey”.

The next common regex is the scan method. Which looks like this: .scan(/…/). This scan method will spit the string by each three characters. The /…/ means all characters including spaces. However if you don’t want to have whitespaces and only letters, numbers, and symbols; the same thing could be done with /\S\S\S/. You also can us /\d/, which will search for numbers. For example:

“You have 100 dollars”.scan(/\d+/)
>// 100

This will return 100 and because if you add the + to the “d” inside the regex it means that it can be any length of digits.

“hello world”.scan(/[alw]/) for a character classes
>// l l w l
“hello world”.scan(/[a-d]/)
>// d

This should be self-explanatory as this will scan the range.

The next is match. For example:

puts “Telephone number cannot contain letters” if “555-555-555q” .match(/[a-z]/)
>// Telephone number cannot contain letters

If the above telephone number contains any of the range in the regex you will receive the error message like above because there is a “q” at the end. Hopefully now you have a decent understanding about Regular Expressions and can use them for themselves.
